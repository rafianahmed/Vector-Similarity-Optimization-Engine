Color Similarity ML Engine

A single-file GitHub Pages app that represents colors as vectors, computes distance metrics, visualizes them, and runs a Bayesian-inspired optimization engine to reduce validation error on a similarity-classification task.

The app generates a labeled dataset of color pairs, extracts vector features, and grid-searches over feature weights plus a probability threshold to minimize validation classification error. That is a practical surrogate for “Bayes error minimization.”
