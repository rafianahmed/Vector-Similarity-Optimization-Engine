What this app does

It turns each uploaded image into a vector representation. Then it compares the target image against the uploaded gallery using cosine similarity, Euclidean distance, and histogram overlap. A Bayesian-inspired calibration layer learns the best weighting and threshold from the uploaded set, so the final ranking is more robust than raw cosine similarity alone.

Real-world use: visual search, duplicate detection, product matching, media retrieval, image recommendation, and content-based image search.
